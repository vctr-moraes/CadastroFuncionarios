## Cadastro de Funcionários

Esta é uma simples aplicação web, escrita em Java, a qual implementa um CRUD, expondo endpoints para comunicação com banco de dados. A aplicação utiliza o framework Spring, com o gerenciador de pacotes Maven.

### Funcionalidades
* Cadastrar funcionário
* Atualizar funcionário
* Buscar funcionário por ID
* Listar todos os funcionários
* Excluir funcionário

### Tecnologias
* Java 17
* Spring Boot 4.0.3
* Maven
* PostgreSQL
